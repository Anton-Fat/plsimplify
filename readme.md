

# PLSIMPLIFY – AutoCAD AutoLISP Script

Скрипт `PLSIMPLIFY` предназначен для упрощения полилиний в AutoCAD.  
Он находит вершины, в которых угол между соседними сегментами меньше заданного, и предлагает удалить такие вершины.

Это полезно для:
- очистки импортированных DXF
- упрощения топографических линий
- удаления лишних точек после экспорта из GIS
- уменьшения количества вершин в длинных полилиниях

---

# Файлы

В проекте используется один файл:

plsimplify.lsp

Рекомендуется хранить файл в папке, которая добавлена в **Support File Search Path** AutoCAD.

Пример:

```
C:\AutoCAD\LISP\plsimplify.lsp
```

Если такой папки нет, её можно создать.

---

# Добавление папки в AutoCAD

1. Откройте AutoCAD
2. Введите команду:

```
OPTIONS
```

3. Перейдите на вкладку:

```
Files
```

4. Найдите раздел:

```
Support File Search Path
```

5. Нажмите **Add**
6. Добавьте папку, где лежит `plsimplify.lsp`

Пример:

```
C:\AutoCAD\LISP
```

7. Нажмите **OK**

---

# Загрузка скрипта

Есть два способа.

## Способ 1 — через APPLOAD

1. Введите команду:

```
APPLOAD
```

2. Найдите файл:

```
plsimplify.lsp
```

3. Нажмите **Load**

Теперь команда доступна в текущей сессии AutoCAD.

Команда для запуска:

```
PLSIMPLIFY
```

---

## Способ 2 — автоматическая загрузка

Чтобы скрипт загружался автоматически при запуске AutoCAD:

1. Откройте `APPLOAD`
2. Внизу окна найдите:

```
Startup Suite
```

3. Нажмите **Contents**
4. Добавьте файл:

```
plsimplify.lsp
```

Теперь скрипт будет загружаться при каждом запуске AutoCAD.

---

# Использование

1. Введите команду:

```
PLSIMPLIFY
```

2. Выберите полилинию
3. Введите минимальный угол (в градусах)
4. Скрипт покажет количество найденных вершин
5. Подтвердите удаление

Пример:

```
Command: PLSIMPLIFY
Выберите LWPolyline:
Введите минимальный угол (градусы): 5
Найдено вершин: 184
Удалить их? (Y/N): Y
```

# Создание кнопки на панели инструментов

Можно создать кнопку для быстрого запуска команды.

## Способ 1 — через CUI

1. Введите команду:

```
CUI
```

2. В разделе **Customizations in All Files** найдите:

```
Toolbars
```

3. Создайте новую панель инструментов:

```
New Toolbar
```

Например:

```
LISP Tools
```

4. В списке **Command List** нажмите:

```
Create a new command
```

5. Заполните параметры:

**Name**

```
PLSIMPLIFY
```

**Macro**

```
^C^CPLSIMPLIFY
```

**Description**

```
Simplify polyline by removing vertices with small angles
```

6. Перетащите команду в созданную панель инструментов.

7. Нажмите **OK**.

Теперь появится новая кнопка.

---

# Создание кнопки на Ribbon (опционально)

Через `CUI` можно также добавить кнопку на Ribbon:

1. В `CUI` откройте:

```
Ribbon → Panels
```

2. Создайте новую панель
3. Добавьте созданную команду `PLSIMPLIFY`
4. Добавьте панель в нужную вкладку Ribbon
